# Hello-World-Node

## Description

Exemplo de app Node

## Installation

1.  Clone the repository: `git clone https://github.com/YourUsername/Hello-World-Node.git`
2.  Navigate to the repository directory: `cd Hello-World-Node`
3.  Install dependencies (if any): `npm install`
4.  Run the application: `npm start`

## Usage

1.  **File: app.js**  - This file contains the core application logic.  It handles basic node app initialization and a simple hello world output.
2.  **File: package.json** - Defines the project's dependencies and scripts.
3.  **File: renovate.json** -  Configures the development environment.
4.  **File: .vscode** -  Configures the editor, terminal, and other VS Code settings.

   -  Run `npm start` from the project directory to execute the application.
